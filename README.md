# Game-2048

### Requirements

* C++ compiler (e.g. `g++`)
* Platform Support:
  * Linux
  * MacOS
  * Windows

### Installation

Open your terminal and clone this project:
```sh
git clone https://github.com/anurag1504t/Game-2048.git
```
Enter the project directory:
```sh
cd Game-2048
```

---

#### Building with g++ Compiler
Build the executable
```sh
g++ game.cpp -o game
```

Run executable
```
./game
```
